---------Installing Node & NPM on EC2 machine------------------------
curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt install nodejs
node --version
npm --version
-------------------Installing GIT on EC2 machine----------------------
sudo apt update
sudo apt install git
git --version
